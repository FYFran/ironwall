## 当前任务

Brain A提出了OWASP Python Benchmark测试方案。以下是他的完整方案：

### 方案概要
- 对OWASP Python Benchmark（1000+ .py测试用例，每个标记vulnerable/safe）跑ironwall scan
- 输出JSON格式: {findings: [{file_path, cwe, category, severity, line_number, ai_confidence}]}
- 与expectedresults.xml对比，per-file匹配
- 计算Precision/Recall/F1/F3

### Brain A的具体决策
1. 跑两轮：纯扫描器baseline + --ai增强版，对比提升幅度
2. Per-file粒度匹配（不是per-line，因为benchmark只有文件级标注）
3. 先50文件试跑验证pipeline，再跑1000全量
4. 写CWE映射表: {"sqli": "CWE-89", "cmdi": "CWE-78", "xss": "CWE-79", "pathtraver": "CWE-22", "weakcrypto": "CWE-327"}
5. 目标指标：Precision>=0.70, Recall>=0.60, F3>=0.62

### Ironwall架构
- 8步管线: gitleaks -> semgrep -> gosec -> hardcoded -> kics -> syft/grype -> database -> supply chain
- AI引擎: 9规则Go离线引擎 + DeepSeek Analyst (OBSERVE->TRACE->VERIFY->ASSESS)
- 命令: ironwall scan [target] --format json --ai
- AI mode需要DEEPSEEK_API_KEY，调用DeepSeek API
- scan整个目录（不是逐文件），一次产出所有findings

### 上次共识指标
- Precision>=0.70, Recall>=0.60, F3>=0.62
